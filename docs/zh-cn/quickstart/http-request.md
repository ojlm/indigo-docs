### 创建一个简单的请求
![](./gif/start.gif)

#### 1. 首先进入自己的项目,如：假设在 `indigo/docs` 项目下面.
![](./images/start.png)

#### 2. 填写 URL `www.baidu.com`,并点击发送按钮获得响应
![](./images/url.png)

#### 3. 填写用例名称,点击 `保存` 就创建完成
![](./images/save.png)

#### 4. 创建完成后会自动生成一个ID,继续编辑会自动保存到这个用例上,另存为则生成新的用例请求
![](./images/saved.png)
